Git

Git

  • 增加文件到暂存区

    git add filename / dir / .

  • 提交暂存区到仓库

    git commit -m [message]

    git commit -v

  • 列出所有本地和远程分支

    git branch -a

  • 列出远程分支

    git remote -v

  • 切换到指定分支

    git checkout [branchname]

  • 上传本地指定分支到远程仓库

    git push [remote] [branch]

流程：

add -> commit -> checkout -> push
